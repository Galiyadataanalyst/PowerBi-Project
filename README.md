## Задача

Компания фиксирует снижение продаж и хочет определить возможные причины. Необходимо построить 
интерактивный аналитический отчёт в Power BI, который позволит оценить ключевые бизнес-показатели,
проверить возможные причины снижения продаж и выявить ограничения имеющихся данных.

## Решение

* Проведён первичный анализ структуры и качества данных в базе MySQL.
* Проверена согласованность таблиц заказов, продаж и товаров. В ходе анализа выявлены несоответствия в суммах и датах между
* отдельными таблицами, а также отсутствие данных, необходимых для расчёта прибыли и маржинальности.
* Построена модель данных в Power BI с установлением связей между таблицами.
* Разработаны DAX-меры для расчёта основных показателей: выручки, количества заказов, среднего чека, количества клиентов,
* доли возвратов и других бизнес-метрик.
* Создан интерактивный дашборд, включающий разделы по продажам, складам, клиентам и персоналу.
* Выполнен анализ динамики выручки, среднего чека и количества заказов по месяцам.
* Проверены гипотезы, связанные с влиянием клиентской базы, регионов, городов, статусов заказов и складских остатков на снижение продаж.
* Проведена оценка ограничений исходных данных и определено, какие бизнес-показатели невозможно рассчитать без дополнительной информации.

## Результат

* Выявлено снижение выручки после октября, сопровождающееся уменьшением среднего чека и количества заказов.
* Проверенные гипотезы, связанные с регионами, городами, статусами заказов и складскими остатками, не подтвердили
* влияние на снижение продаж.
* Обнаружены ограничения исходных данных, препятствующие корректному расчёту прибыли, маржинальности и фактических логистических затрат.
* Подготовлен интерактивный Power BI-отчёт, позволяющий отслеживать ключевые показатели компании и использовать его как основу
* для дальнейшего анализа после получения дополнительных данных.
